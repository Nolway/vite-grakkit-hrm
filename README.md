# vite-grakkit-hrm

Vite plugin for HRM with [Grakkit](https://github.com/grakkit/grakkit).


## Requirements
- Node < v20
## Installation

Add the dependency:

```bash
  npm install vite-grakkit-hrm
```


## Used By

This project is used by the [Grakkit Boilerplate project](https://github.com/Nolway/grakkit-boilerplate).
